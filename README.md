## STELLAR WARP

Un juego arcade que hicimos para practicar git en equipo. Controlas una nave, le disparas a todo lo que se mueva y tratas de no morir antes de llegar al jefe final.

## ¿Cómo se juega?

Mueves la nave con `WASD` o las flechas, disparas con `SPACE` y si juntaste suficiente valor, el disparo especial es con `Z`. Si necesitas respirar, `P` pausa el juego.

Son 4 oleadas en total. Las primeras tres son asteroides y enemigos normales, pero la cuarta... ahí aparece el **OMEGA TITAN** y las cosas se ponen serias.

Por el camino van cayendo power-ups: corazones para recuperar vida, velocidad, fuego rápido y escudo. Agárralos todos, los vas a necesitar.

## El proyecto tiene 3 archivos

- `index.html` — las pantallas del juego
- `styles.css` — todo el estilo retro
- `app.js` — donde vive la lógica
